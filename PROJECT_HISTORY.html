🎯 核心功能

1. 單檔股票分析

  • K 線圖與移動平均線 (MA20/MA50/MA200)
  • 技術指標：RSI、MACD、布林帶、隨機指標、ATR
  • 基本面分析：P/E、P/B、ROE、ROA、負債比、毛利率、淨利率
  • 歷史數據：上市 10 年 + 上櫃近 6 個月完整日資料

2. 族群比較分析

  • 產業內對比、產業領頭股、漲跌排行
  • 相關性分析、營收趨勢圖

3. 選股篩選

  • 預定義篩選：看漲信號、超賣/超買股票、成交量大漲股
  • 自訂篩選：價格範圍、成交量、技術指標、產業類別
  • 批次篩選：P/E、ROE、市值、殖利率

4. 族群分析（標籤系統）

  • 自訂標籤、多對多映射、即時持久化
  • 批次操作、搜尋式股票選擇器

5. 量化策略回測

  • SMA 交叉策略、RSI 策略、MACD 策略
  • 完整績效指標：總報酬率、年化報酬率、最大虧損、勝率
  • 除權息調整、多檔組合回測

6. 投資組合管理 (P4-1)

  • 組合價值追踪、權重分配分析、績效分析
  • 再平衡建議、淨值曲線圖

7. 自動化報告生成 (P4-3)

  • 市場摘要、股票報告、產業報告、篩選結果報告
  • 週報/月報、CSV/HTML 格式匯出

🎨 設計原則

  • 🏗️ 分層隔離 (Layering) - 數據訪問層屏蔽存儲細節，分析層獨立於 UI
  • 📈 增量優先 (Incremental by Default) - 預設使用增量 ETL，秒級完成
  • 💾 雙存儲策略 (Dual Storage) - SQLite 維度資料 + Parquet 時間序列
  • 🔓 開放封閉 (Open-Closed) - 對擴展開放，對修改封閉
  • 🔄 DRY (Don't Repeat Yourself) - 統一的數據查詢 API，共享的技術指標

⚡ 性能優化

數據結構

  • Parquet: 列式存儲，壓縮率高，查詢快
  • SQLite: 輕量級關係數據庫，便於複雜查詢
  • ETL_Status: 避免重複讀取所有 JSON

增量處理

  • 日常更新只處理新 JSON（1-2 個檔案）
  • 不重新寫入所有 11,718 個 parquet
  • ETL_Status 表追蹤已完成的工作

查詢優化

  • 篩選時使用集合運算
  • 批量讀取 Parquet 檔案
  • 日期範圍預過濾

計算優化

  • 技術指標使用向量化操作 (Pandas/NumPy)
  • 避免 Python 循環
  • 緩存常用計算結果

🔒 安全與可靠性

TPEx SSL 處理

  • 櫃買中心憑證驗證失敗 → 使用 verify=False
  • urllib3.disable_warnings() 抑制警告
  • DNS 錯誤自動重試（最多 3 次，指數退避）

數據驗證

  • JSON 格式檢查（TWSE fields9/data9 vs tables 陣列）
  • 欄位名稱模糊匹配（代號 → 證券代號）
  • 空值/異常值處理

錯誤處理

  • 連續 5 次 API 錯誤自動中斷
  • 重試機制（3 次，含退避）
  • 檔案不存在、格式錯誤的優雅降級

# 分析單檔股票 from core.analysis import StockAnalyzer analyzer = StockAnalyzer("2330") indicators = analyzer.get_latest_indicators() # 篩選股票 from core.filters import StockFilter result = (StockFilter() .filter_by_price_range(100, 500) .filter_by_rsi(max_rsi=70) .execute()) # 回測策略 from backtesting.engine import BacktestEngine, StrategyLibrary engine = BacktestEngine("2330") engine.add_signal(StrategyLibrary.sma_crossover_strategy) results = engine.backtest() # 投資組合管理 (P4-1) from portfolio import Portfolio portfolio = Portfolio() portfolio.add_stock("2330", 0.5) portfolio.add_stock("2454", 0.5) # 報告生成 (P4-3) from report import ReportGenerator report_gen = ReportGenerator() report_gen.generate_market_summary()

⚠️ 免責聲明

本工具僅供分析參考，不構成投資建議。

股票投資存在風險，應進行充分的風險評估。過去的績效不代表未來的表現。

使用者應自行承擔投資決策的風險與責任。
